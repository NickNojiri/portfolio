Relay

A/B-test AI prompts in production and see which one actually wins.

Teams tweak prompts constantly, but which wording is better? Relay lets you author a system prompt, ship it through a gateway, split traffic with feature flags, and track cost and latency per version.

Engineering decisions

  • Polyglot monorepo: a Rust flag engine for low-latency traffic splitting, a Python/FastAPI gateway for the model ecosystem, a TypeScript dashboard.
  • Wired with pnpm + Turborepo + Cargo so one command builds three languages.
  • Per-version cost and latency tracking, so "which prompt wins" is a number, not a hunch.

act 02 Live · 143 tests

SpotBot

Turns the Instagram reels your friends share into places you actually go.

Your group chat is full of "we should go here!" reels nobody acts on. SpotBot reads the caption and the spoken audio, figures out the venue, time, and location, and posts a votable card in Discord.

Engineering decisions

  • Local-first: Ollama for the LLM and OpenStreetMap for geocoding — no per-call API cost, runs offline.
  • faster-whisper transcribes reel audio, so a spot mentioned only out loud still gets captured.
  • 7-stage ingestion pipeline (fetch → refine → geo → temporal → sink → recommend) with 143 offline tests.

act 03 In progress

Resume Improver

Scores a resume, then rewrites weak bullets with grounded suggestions.

Built on an MIT resume-to-score pipeline, extended into a full-stack improver: it extracts structured data from a PDF, scores it, then streams grounded rewrites of weak bullet points — with guards so it never invents experience you don't have.

What's being built

  • Grounded rewrites: every suggestion is anchored to the original resume, with hallucination guards.
  • FastAPI + Server-Sent Events stream the rewrite token-by-token into a React diff view.
  • Exports a polished DOCX so the improved resume is ready to send.
